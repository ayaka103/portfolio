# 見出し分析ガイド

## 分析観点

### 1. 構造パターン分析
- **見出し階層の深さ**: H1〜H4のどこまで使われているか
- **見出し数**: 全体のセクション数、H2/H3の比率
- **構成パターン**: 導入→本論→まとめ、問題提起→解決策、リスト型 等

### 2. トピックカバレッジ分析
- **共通トピック**: 3記事中2記事以上で扱われているトピック（必須トピック）
- **独自トピック**: 1記事のみが扱うトピック（差別化のヒント）
- **欠落トピック**: どの記事も扱っていないが重要なトピック（差別化チャンス）

### 3. SEO最適化チェック
- **キーワード配置**: H1/H2にターゲットキーワードが含まれるか
- **検索意図との整合**: 情報型/取引型/ナビゲーション型の意図に合った構成か
- **見出しの具体性**: 曖昧な見出し vs 具体的・数値入りの見出し
- **文字数バランス**: 見出しが長すぎず短すぎず（15〜40文字目安）

### 4. 差別化戦略
- **網羅性で勝つ**: 競合が薄いセクションを深掘り
- **独自セクション追加**: 競合にない切り口を追加
- **構成順序の最適化**: ユーザーの思考フローに沿った順序
